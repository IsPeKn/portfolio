Proyecto HTML-CSS de repaso

Proyecto HTML-CSS : Descripción paso a paso

  1. Creación del diseño básico en un folio a mano. Hay herramientas actualmente para hacer diseños, pero era un esquema incial.
  2. Primero se crea la estructura. Iniciamos con el index.html (página principal con menú, el cuál irá en el resto de páginas). Creamos 2 páginas más para las asignaturas. Intento hacer el menú lateral y de navegación (posible cambio a nav vertical), o dejarlo en horizontal “sticky”
  3. Creación y fijación del menú de navegacion para todas las web añadidas. Con su CSS y HTML.
  4. Hago un rediseño de las webs y portada. Por ello usé otro IDE: WebStorm.
  5. Inlcuyo los subapartardos y los estilos de estos.
  6. Revisar las rutas de la web e instalación de browser-sync para una navegación fluida.
  7. Los commits realizados se hacen al finalizar el día o mejoras importantes en la web.
  8. Se sube el proyecto a GitHub Pages y se comprueba que navega bien con otros navegadores, equipos y pantallas con diferentes pulgadas.
  9. Finalmente se intenta trabajar con los media queries para ajustar nuestra web a dispositivos móviles (tanto en horizontal como en vertical).
